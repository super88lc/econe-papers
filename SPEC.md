# Econe Papers - 经济学论文精选

## 1. 项目概述

**目标**：每日自动抓取 arXiv.org 最新经济学论文，AI 筛选归类，展示选题新颖、研究水平高的 10 篇论文。

**核心价值**：帮助经济学研究者快速跟踪前沿研究，无需手动浏览大量论文。

**用户画像**：经济学研究者、学者、在读研究生

---

## 2. 功能规格

### V1 (MVP)

| 功能 | 描述 |
|------|------|
| 定时抓取 | 每日自动抓取 arXiv q-fin/EC/GN/PM |
| AI 阅读 | LLM 提取标题、作者、摘要、关键词、研究主题 |
| 分类归类 | 宏观、微观、计量、金融、行为、实验等 |
| 精选 10 篇 | AI 评分排序 |
| 流式首页 | 按日期分组展示，无限滚动加载历史 |
| 每日更新 | 每天自动更新 |

### V2

全文精读、搜索功能、收藏、邮件订阅

---

## 3. 技术架构

```
arXiv.org → Python (抓取+AI) → JSON → Next.js → Vercel
```

- 后端：Python 3.x + arXiv API + MiniMax API
- 前端：Next.js + Tailwind CSS + Vercel
- 数据：JSON 文件

## 5. 数据模型

```typescript
interface Paper {
  id: string;           // arXiv ID
  title: string;        // 论文标题
  authors: string[];   // 作者列表
  abstract: string;     // 原文摘要
  categories: string[]; // arXiv 分类
  published: string;   // 发布日期
  updated: string;      // 更新日期
  pdfUrl: string;       // PDF 链接
  
  // AI 处理后
  chineseTitle: string;    // 中文标题
  chineseAbstract: string; // 中文摘要
  researchField: string;   // 经济学子领域
  keywords: string[];     // 关键词
  scores: {
    overall: number;      // 综合评分 0-10
    novelty: number;      // 选题新颖度 1-5
    quality: number;      // 研究水平 1-5
    readability: number;  // 可读性 1-5
  };
  summary: string;        // 一句话总结
}

interface DayPapers {
  date: string;        // 2026-03-01
  papers: Paper[];     // 当日精选论文
  total: number;       // 篇数
}
```

## 6. API 设计

| 端点 | 描述 |
|------|------|
| GET /api/papers?date=2026-03-01 | 获取指定日期论文 |
| GET /api/papers?before=2026-02-28&limit=30 | 获取历史论文(分页) |
| GET /api/papers/[id] | 论文详情 |
| GET /api/dates | 获取有数据的日期列表 |

---

## 7. 交互设计

| 交互 | 描述 |
|------|------|
| 滚动到底部 | 自动加载前一天的数据 |
| 点击论文 | 展开/收起摘要 |
| 分类 Tab | 筛选当天论文 |
| PDF/arXiv | 新窗口打开 |
